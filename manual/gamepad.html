Ґеймпад

Pilas Engine забезпечує доступ до зовнішніх елементів керування, на зразок ґеймпада, щоб ви могли зробити ігри набагато інтерактивнішими.

Кожного разу, коли ви підключаєте ґеймпад, ігровий рушій розпізнає його та створює внутрішній об’єкт, щоб ви могли отримати доступ до кожної з кнопок і елементів керування ґеймпада.

Підтримувані типи ґеймпадів

Внаслідок взаємодії з деякими загальнодоступними функціями вебпереглядача для доступу до ґеймпадів, підтримуються не всі типи елементів керування.

Ґеймпади, які були практично протестовані, наведені нижче:

  • gamepad xbox360 sobre windows 10 (як через дротовий USB, так і через бездротовий приймач USB).
  • pro controller de nintendo switch (bluetooth).
  • joy-con de nintendo switch (bluetooth).

Як ви бачите, список не дуже довгий, багато ґеймпадів не працюють із сучасними вебпереглядачами, а інші сучасні ґеймпади, як-от ґеймпади для PS4, ще не тестувалися (але вони, напевно, будуть працювати).

Рекомендуємо вам провести тест перед початком програмування: запустіть кілька прикладів, призначених для тестування ґеймпадів, і перевірте, чи ваш ґеймпад правильно працює.

Доступ до елементів керування

Щоб керувати актором через ґеймпад, вам не потрібно робити нічого особливого. Якщо підключення ґеймпада до ігрового рушія було вдалим, ви зможете отримати доступ до нього так:

this.control.izquierda
this.control.derecha
this.control.arriba
this.control.abajo
і т. д.

Це ті ж властивості, що використовуються і для клавіатури.

Ці властивості можна запитувати з коду актора таким чином (зверніть увагу, що ви пишете цей код у функції actualizar):

if (this.control.izquierda) {
  this.x -= 5;
}

if (this.control.arriba) {
  this.y += 5;
}

// etc.

Однак, може статися так, що ви захочете отримати доступ до всіх функцій ґеймпада, наприклад до того, що вони мають два аналогових джойстика або кілька кнопок. У такому разі вам слід отримати доступ до елементів керування через змінну this.pilas.control.pad_1.

Наприклад, щоб поступово пересувати персонажа за допомогою лівого аналога, ви повинні використати такий код:

this.x += this.control.gamepad_1.analogico_izquierdo_x * 10;
this.y += this.control.gamepad_1.analogico_izquierdo_y * 10;

Тут немає необхідності використовувати умову типу if, оскільки властивості analogico_izquierdo_x і analogico_izquierdo_y повертають число між -1 та 1, якщо елемент керування рухається в певному напрямку або 0, якщо він знаходиться в стані спокою.

Для правого аналога також існують властивості, що називаються analogico_derecho_x та analogico_derecho_y.
