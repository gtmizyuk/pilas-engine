Поведінка

При розробці ігор зручно мати спосіб повідомити акторам певне завдання, яке вони повинні виконати.

У цьому раі використовується поняття поведінки. Поведінка - це об'єкт, який символізує дію, яку повинен виконати актор.

Наприклад: охоронець у грі може ходити туди-сюди по коридору:

  • йти ліворуч до кінця коридору
  • зробити повний поворот
  • йти праворуч до кінця коридору
  • зробити повний поворот
  • і т. д.

У цьому випадку поведінка містить 4 дії, а завданням є охорона коридору. За потреби, ми можемо контролювати таку поведінку, наприклад, повторювати її чи ні.

Реалізація поведінки

Щоб вказати актору на виконання певної поведінки, ми повинні викликати функцію hacer, зазначивши назву поведінки:

let mi_actor = pilas.actores.actor();
mi_actor.hacer("aparecer");

Зауважте, що різні поведінки можна об'єднувати, щоб отримати ефекти або анімацію, що відбуваються по черзі. Наприклад, якщо ми хочемо показати або приховати актора, ми можемо зробити це так:

let mi_actor = pilas.actores.actor();
mi_actor.hacer("aparecer");
mi_actor.hacer("desaparecer");

Деякі поведінки отримують параметри для швидкості, позиції тощо.

Наприклад, якщо ми хочемо, щоб актор зникав і з’являвся кілька разів дуже швидко, ми можемо написати щось подібне:

let mi_actor = pilas.actores.actor();

for (i=0; i<50; i++) {
  mi_actor.hacer("aparecer", {velocidad: 10});
  mi_actor.hacer("desaparecer", {velocidad: 10});
}

Вбудована поведінка

Це список містить вбудовані типи поведінки:

  • desaparecer (параметри: velocidad) - зникнути (параметри: швидкість)
  • aparecer (параметри: velocidad) - з'являтися (параметри: швидкість)
  • eliminar (без параметрів) - видалити (без параметрів)
  • mover (параметри: x, y, demora) - переміщення (параметри: x, y, затримка)

Спеціальна поведінка

Якщо ви хочете створити власну поведінку, ви можете підійти до цього наступним чином.

По-перше, ви повинні створити клас, який успадковує від Comportamientos і мати принаймні один такий метод запуску й оновлення:

class MiComportamiento extends Comportamiento {

  iniciar(argumentos) {
  }

  actualizar() {
    // повернути true, щоб зупинити поведінку.
  }

  terminar() {}
}

Тоді ви повинні зв’язати його з модулем поведінки:

pilas.comportamientos.vincular("mi comportamiento", MiComportamiento);

і, нарешті, призначити його акторові ("mi comportamiento" - моя поведінка):

mi_actor.hacer("mi_comportamiento");
